# Installation and Usage:
## Requirements:
- Python 3.9 or higher
- Poetry
## Installation:
### 1. Clone the repository:
git clone https://github.com/vvladalvv/python-project-50.git
### 2. Install dependencies using poetry:
poetry install
## Usage:
### 1. Run:
gendiff <path_to_file1> <path_to_file2>
### 2. For help, use:
gendiff -h
